## Project Info

1. Wrote some python to **separate the PDFs into smaller PDFs**, each containing only a **single question**.
2. Did the same for the markscheme.
3. Converted those PDFs to `.png` format. (_Styling the PDFs was very inconvenient._)
4. Whipped up a simple frontend to **pair question + answer**, and **sort all questions by week!**
5. Slapped it onto a website.
